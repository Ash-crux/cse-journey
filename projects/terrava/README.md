# Terrava — Demo Commercial Website

**Live URL:** https://terrava-earthly-elegance.lovable.app  
**Built:** June 2026  
**Type:** Full-stack commercial website (demo/practice project)

## What This Is
A demo natural lifestyle brand website built as practice before 
building the real Velmora client project.

## Tech Stack
- React (via Lovable AI builder)
- Tailwind CSS
- Supabase (PostgreSQL database)
- Deployed on Lovable

## What I Built
- Full homepage with hero, category cards, product grid
- Testimonials, newsletter, footer sections
- Supabase database with 4 tables: products, customers, orders, order_items
- 6 products seeded into the database
- Live deployment on lovable.app

## Database Tables
- products — stores all product info
- customers — stores customer details
- orders — stores order records
- order_items — links orders to products
